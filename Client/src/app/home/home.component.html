Watchlist App

Welcome to the Watchlist Application. You must login to be able to use the application.

  • You can create multiple watchlists

  • You can add multiple titles to each watchlist

  • You can select whether a title on the watchlist has been watched or not

  • Items in the watchlist are originally ordered by creation date

  • There is no save button, each watchlist changes when a title has been added or removed, or a title's watche checkbox is selected or deselected
